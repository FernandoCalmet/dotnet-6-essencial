# CLI de DOTNET 6

## Parámetro de ayuda

```bash
dotnet --help
```

## Creación de proyectos

```bash
dotnet new
dotnet new --list
#templates
dotnet new --search microservice
dotnet new --search iot --language F#

dotnet new console
dotnet new webapi
dotnet new web -o MinApi
```

## Estado del SDK

```bash
dotnet sdk
dotnet sdk check
```

## Workload

```bash
dotnet workload
dotnet workload search maui
```

## Ejecución de un proyecto

```bash
dotnet run
```
